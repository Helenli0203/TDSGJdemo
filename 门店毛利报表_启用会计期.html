优免额 = 折前销售额 - 折后销售额。

反应优惠免单的总金额

理论成本 = 原料平均采购入库单价 *  原料理论耗损量。

如果查询时间段内没有采购入库发生，平均采购入库单价取采购入库发生时间离查询起始日期最近的一次采购入库单价。


实际成本= 原料平均采购入库单价 * 原料实际耗用量

实际耗用量 = 期初库存 + 期内入库量 - 期末库存 - 期内调拨出库量 - 期内退货出库量

成本差分额 = 实际成本 - 理论成本

反应原料浪费赞成的金额损失。

成本差分率 = (实际成本 - 理论成本) / 理论成本。

反应原料浪费率。

折前理论毛利 =  折前销售额 - 理论成本

反应去除优惠免单及原料浪费的理想毛利。

折后理论毛利 = 折后销售额 - 理论成本

反应优惠免单后去除原料浪费外的理想毛利。

折前理论毛利率 = 折前理论毛利 / 折前销售额反应去除优惠

反应去除免单和原料浪费外的理想毛利。

折后理论毛利率 = 折后理论毛利 / 折后销售额

反应优惠免单后去除原料浪费外的理想毛利。

折前实际毛利率 = 折前实际毛利 / 折前销售额

反应去除折扣损失的实际毛利。可作为考核后厨的指标。

折后实际毛利率 = 折后实际毛利 / 折后销售额

反应最终的毛利率结果, 门店实际实际得到的毛利率。

毛利折损金额 = 折前理论毛利 - 折后实际毛利

反应实际毛利与最理论毛利的差距。

折前实际毛利 = 折前销售额 - 实际成本

反应去除折扣损失的实际毛利，可作为考核后厨的指标。

折后实际毛利 = 折后销售额 - 实际成本

反应实际毛利

毛利折损率 = 折前理论毛利率 -  折后实际毛利率

反应实际毛利率与最理论毛利率的差距。
